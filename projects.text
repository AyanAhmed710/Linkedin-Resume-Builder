Project 1: Customer Churn Prediction (Telecom – Machine Learning)
Category: Machine Learning
Industry: Telecom
Tools Used: Python, Pandas, NumPy, Scikit-learn, XGBoost, SMOTE, Matplotlib
Objective:
Developed a predictive machine learning model to identify telecom customers likely to churn, enabling proactive retention strategies.
Dataset:
Telco Customer Churn dataset including demographic, billing, and service usage information.
Methods & Approach:
Performed extensive data preprocessing including missing value handling, encoding categorical variables, and feature scaling.
Applied feature engineering to extract customer behavior indicators.
Handled class imbalance using SMOTE.
Trained multiple models including Logistic Regression, Random Forest, and XGBoost.
Optimized hyperparameters using GridSearchCV.
Evaluated performance using ROC-AUC, Precision, Recall, and F1-score.
Results & Impact:
Achieved ROC-AUC score of 0.87.
Identified high-risk customers, enabling targeted campaigns that reduced churn by 18%.
Skills Demonstrated:
Machine Learning, Classification, Feature Engineering, Model Evaluation, Imbalanced Data Handling.
